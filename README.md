# Simulation setup for my PHD

Simulations shown in the _Analysis Spatial Heterogeneity using Optimal Transport_ paper co-authored by Xavier Bacon are present at [optimal_transport/results/article/Paris/](./optimal_transport/results/article/Paris/), with the code used to generate it present at [optimal_transport/generate_fig_article.py](./optimal_transport/generate_fig_article.py) (_including the links to all of the data, as the databases were too large to be included in this repository_). Additionaly the librairy used to complute theses results is open-source and can be found at the link [github.com/jolatechno/ot-heterogeneity](https://github.com/jolatechno/ot-heterogeneity) or as a submodule of this repository located at [optimal_transport/ot-heterogeneity/](./optimal_transport/)
